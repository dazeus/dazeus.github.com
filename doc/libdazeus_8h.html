Get the internal file descriptor of this dazeus instance. Do not read or write from/to this socket. This method can be used to add the given file descriptor to an event loop (i.e. using select()). Returns 0 if no connection was made, or it has recently failed.

dazeus_event* libdazeus_handle_event ( dazeus *  d,
int  timeout 
)

Return the next event received (in order of coming in). Don't forget to free the resulting structure using libdazeus_event_free. Do NOT libdazeus_stringlist_free its 'parameters' property, though. If the 'timeout' parameter is -1, will wait forever until the first event. If the 'timeout' parameter is 0, will try getting an event exactly once. If the 'timeout' parameter is > 0, will try that many seconds to get an event. If this method returns NULL, it means either an error occured, or there was no event inside the given timeout period.
